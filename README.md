# JobBoard Platform

A Full Stack Job Board Platform built using React, Django REST Framework and SQLite.

## Features

- User Registration
- User Login (JWT Authentication)
- Create Job
- View Jobs
- Apply for Jobs
- Resume Upload
- Dashboard
- My Applications

## Tech Stack

Frontend
- React
- Vite
- Tailwind CSS
- Axios

Backend
- Django
- Django REST Framework
- JWT Authentication

Database
- SQLite

## Installation

### Backend

```bash
cd backend

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```

### Frontend

```bash
cd frontend

npm install

npm run dev
```

Frontend runs at

```
http://localhost:5173
```

Backend runs at

```
http://127.0.0.1:8000
```
